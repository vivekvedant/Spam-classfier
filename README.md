

# Spam-classifier

## Required Dependencies
1. Flask
2. Pickle
3. sklearn
4. pandas
5. matplotlib
6. nltk (natural language tool kit)
7. wordcloud
8. re (Regular expression)

## Installing Dependencies
- pip install flask
- pip install pickle
- pip install sklearn
- pip install pandas
- pip install matplotlib
- pip install nltk
- pip install wordcloud
- pip install re
